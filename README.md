# Next Open Jira

This app is a Jira clone using Material UI (dynamic dark/light theme), drag and drop without library and Next API Routes connected to MongoDB.

## Usage

Install dependencies

```shell
yarn
```

Run app locally for development

```shell
yarn dev
```
